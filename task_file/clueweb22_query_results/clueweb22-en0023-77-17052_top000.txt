Sailor moon - Cursors
A great set of anime cursors for fans of the anime series "Sailor moon". The main cursor has a pink color and the usual triangular shape, next to which is the avatar of the character - a girl named "Sailor Moon". There are 17 files in total in the archive. Of these, 15 are the cursors themselves, while the remaining two are a preview and an inf-file for automatic installation on the desktop.
To install cursors « Sailor moon » in the format CUR, ANI unpack the archive and run the executable file install.inf as administrator, as a result of which the pointers are automatically installed on the computer. Then you just have to select them in the system settings. Learn More »
