打ちやすく抜けにくい新発想ペグ
“ 矢 ” をモチーフにして生まれたカタチ
矢（アロー） とは弓などを用いて対象となる物を遠方からでも狙うことができる武器です。 日本 での歴史は縄文時代まで遡り、主に狩猟に用いられていたようです。
時代の移り変わりに合わせて狩猟から戦、神事、弓術等に用途を変え、繰り返し改良されながら 現代に受け継がれて来ました。
そして、ここ 日本 で新たな変貌をとげた、 地面に打ち込む矢 【ARROW】 の歴史が幕を開けます。
基本性能を磨き上げた ARROW の特徴
①打ちやすく、固い地面にぐんぐん刺さる！
②四角い軸と抜け止めが地中でくいつき、強力な固定力を発揮！
③日本刀のように頑丈で、表面はサビや磨耗に強い加工！
十文字の打撃面
あなたの打撃が 的【＋】を射る
視覚効果 で直観的に打つ位置がわかり、打撃面が 広く平ら なため狙いやすい。
また、打ち込む軸が打撃面から 真っ直ぐ延長線上にある ので打撃の 力がしっかり伝わり 、気持ち良く地面に打ち込める。
◇ ◇ ◇ ◇ ◇ 打ち込み性能比較 ◇ ◇ ◇ ◇ ◇
5kgの重りを同じ高さから自由落下させてペグを芝生の地面に打ち込み 、用意した各ペグの軸が地中に打ち込まれた深さを測定しました 。
下記の簡易試験装置を製作し、できるだけ 一定の条件 になるよう準備しました。
※試験は各ペグの先端部35mmを地中に挿した状態をスタートとします
（１） ARROW185mm と他社製品の比較
＜試験条件＞ 地表400mmの高さから5kgの重りを1度自由落下させる。
（２） ARROW285mm と他社製品の比較
＜試験条件＞ 地表750mmの高さから5kgの重りを2度自由落下させる。
上記の結果から ARROW の 打ち込み性能が非常に優れている ことがわかります。
実際はハンマーで打ち込むので、打撃面を外しにくい ARROW は 他社製品との違い をより体感いただけます。
◇ ◇ ◇ ◇ ◇ 抜け止めの効果 ◇ ◇ ◇ ◇ ◇
なぜ ARROW は抜けにくいのか！？
抜け止め形状 が ガイロープからの荷重 を受けて 地球をがっちりキャッチ！
★抜くときは 抜け止め形状 が緩むように（ ガイロープからの荷重 と反対方向へ） ハンマー等で打つと、スキマができるので 抜きやすくなります！
◇ ◇ ◇ ◇ ◇ 熱処理 のプロの技 ◇ ◇ ◇ ◇ ◇
熱処理 したペグ ARROW は本当に 硬く、頑丈 なのか！？
これを検証するため社内の かたさ試験機 で硬度測定を行いました。下のグラフのように表面付近が最も硬く、表面から芯部に向かってやわらかくなっています。
データからわかるように 熱処理 でペグ全体に硬さと粘りが生まれています。
この結果は、ペグが 曲がったり折れたりしにくい特性を持っている ことを示しています。
◎そして、 熱処理後 の金属組織を顕微鏡で観察したところ、内部に欠陥がなく表面に硬化層が形成されています。
熱処理 により、 欠けたり摩耗しにくい状態 になりました！

実行者紹介
はじめまして オカネツ工業株式会社 と申します。
わたしたちは、農業機械・部品の製造を主軸とする、 岡山県 にある創業73年を迎えた企業です。 金属熱処理事業からスタート し、長年のノウハウが蓄積された 熱処理技術 を活かした完成度の高い製品を生み出してきました。
設計から加工、熱処理、塗装、組立、製品出荷に至る各工程が効率的にできるような体制が整っており、お客様のニーズに柔軟に対応できる強みが特徴です。
新たな発想を図面に起こすこと、高精度の部品を作ること、完成品を自らの手で組み上げることなど、 製造業としての仕事が全て同一事業所内にあります 。 この環境を生かし、何か新しいものがつくれないかということで商品企画プロジェクトが発足。
そして、私の趣味であるキャンプで必ず使うものは！？というところからこの企画がスタートしました。多くの方にユーザーヒアリングを行った結果、テント・タープの設営に 鍛造ペグ を使っている方が多く・・・迷ったら 鍛造ペグ を購入する市場の流れを感じました。その理由として完成度の高い 鍛造ペグ より魅力的な商品が存在しないのが現実のようです。
しかし、この市場動向に反して 目新しさや尖がった商品を求める声 をたくさん聴きました。そこで、自身の考えとユーザーの声に基づいてコンセプトを立て、企画を進めて参りました。多くの皆様にご協力いただき、改良を加えながらペグの完成度を高めることに全集中！
ユーザーの目線に立ち ペグの最も大切な性能と見定めた 【打ちやすさ】【抜けにくさ】【頑丈さ】 に 尖がったデザイン（機能美）を融合 させることで完成したのが自信作 ARROW（アロー） です。
今回、応援購入いただいた皆様の相棒として ARROW（アロー） が活躍できるよう、プロジェクトメンバー一同心を込めて対応していきます。
